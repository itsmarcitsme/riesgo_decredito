# Demo Streamlit — Riesgo de Crédito (Probabilidad de Impago)

Esta app permite calcular la probabilidad de impago de un solicitante.

## Ejecutar en local
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Desplegar en Streamlit Community Cloud
1. Sube esta carpeta a un repo de GitHub.
2. En Streamlit Community Cloud → **New app**.
3. Selecciona tu repo y pon **Main file path** = `streamlit_app.py`.
4. Deploy.
